# Blueprint: Agropoli República Centroafricana

## Visión General

Este proyecto es un sitio web estático para Agropoli República Centroafricana, una iniciativa centrada en la generación de créditos de carbono a través de un modelo de desarrollo sostenible en la República Centroafricana. El sitio web tiene como objetivo proporcionar información sobre el proyecto, sus socios y el impacto que busca generar.

## Plan de Desarrollo Actual

El plan de desarrollo actual se centra en mejorar la experiencia del usuario y la organización del código.

### Tareas Completadas

*   **Consolidación de Estilos:** Se han unificado los estilos de todos los archivos HTML en una única hoja de estilos, `style.css`, para mejorar la mantenibilidad y el rendimiento. Se ha eliminado el archivo `estilos.css` redundante.
*   **Adición de Favicon:** Se ha añadido un favicon a todas las páginas HTML para mejorar la identidad visual y la experiencia del usuario.
*   **Añadir Animaciones Sutiles:** Se han incorporado animaciones CSS sutiles para mejorar la interactividad y el atractivo visual del sitio.
*   **Optimización de Imágenes:** Se ha implementado la carga diferida (`loading="lazy"`) en las imágenes para mejorar los tiempos de carga de la página.
*   **Actualización de Enlace:** Se ha actualizado el enlace al registro de Pepitee en la página del proyecto REDD+.

### Próximos Pasos

1.  **Despliegue:** Desplegar el sitio web en un servicio de alojamiento estático para que sea accesible al público.

## Estructura del Proyecto

*   `index.html`: La página de inicio del sitio web.
*   `socio-arioli.html`: Página de detalles del socio Instituto de Investigación Arioli (DECARBONIKA).
*   `socio-blotix.html`: Página de detalles del socio Blotix Fund LLC.
*   `socio-capitalimprese.html`: Página de detalles del socio CAPITALIMPRESE.
*   `proyecto-redd.html`: Página de detalles del Proyecto REDD+.
*   `style.css`: La hoja de estilos principal que contiene todos los estilos del sitio.
*   `404.html`: Página de error 404.
*   `blueprint.md`: Este archivo, que documenta el proyecto.
